## ✨ 核心功能

### 🤖 AI智能识别
直接粘贴抖音、快手的分享文字（包含文字+链接的整段），AI自动提取视频链接，无需手动清理。

### 🎯 全平台覆盖
支持 **抖音、快手、B站、小红书、YouTube、TikTok** 等10+主流平台，持续扩展中。

### 🔒 突破下载限制
即使创作者设置了"不允许下载"，AI快侠也能智能解析并下载视频内容。

### 💬 AI对话助手
内置AI对话功能，不只是下载工具，更是你的智能视频助手。

### 📱 多端支持
- 🍎 iOS / Android 手机端
- 💻 macOS / Windows / Linux 桌面端
- 🌐 浏览器在线使用

## 📖 使用方法

**超简单，三步搞定：**

```
1️⃣ 复制分享文字
   例如：「卷疯了！61 键满配 + RGB 光效... https://v.douyin.com/xxx 复制此链接，打开【抖音】...」

2️⃣ 粘贴到AI快侠
   AI自动识别并提取视频链接

3️⃣ 点击下载
   视频自动保存到本地
```

## ❓ FAQ

**Q: 为什么有些视频下载不了？**
A: 部分视频可能因平台限制或网络问题暂时无法下载，请稍后重试。

**Q: 下载的视频有水印吗？**
A: 大部分平台支持无水印下载，部分平台因技术限制可能保留水印。

**Q: 需要付费吗？**
A: 基础功能免费使用，高级功能需要充值积分。

**Q: 支持批量下载吗？**
